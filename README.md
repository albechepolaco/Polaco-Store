# Polaco-Store
A POLACO STORE é uma loja virtual moderna focada em moda urbana e sneakers exclusivos, unindo estilo, conforto e autenticidade. Desenvolvida como projeto de portfólio, demonstra habilidades em front-end, com carrinho interativo, filtros por categoria, modo claro/escuro e uso de LocalStorage para uma experiência dinâmica e profissional.
